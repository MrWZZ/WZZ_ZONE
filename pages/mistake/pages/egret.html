Egret

数据容器中的皮肤需要提前加载，否则在使用时候在加载的话会导致加载过慢，属性没有附上。

有时候报某一个控件未定义错误，很大可能是因为皮肤没有加载完成引起的。这时候要做一个初始化方法，判定控件时候初始化完成。

在监听touch_begin、touch_end、touch_move的事件中，可能因为移动太快，move事件跟不上。
在抬起的时候会触发TOUCH_RELEASE_OUTSIDE事件，导致touch_begin、touch_end不能成对触发。

在获取舞台宽、高的时候，直接通过this.stage.width/height获取到的是零。
需要通过egret.MainContext.instance.stage来获取。

遇到页面没有正常调用方法、没有正常开启或关闭的时候，先想想有没有可能是页面还没有初始化完毕。

储存信息到本地

// 存储
var key:string = "bestscore";
var value:string = "95";
egret.localStorage.setItem(key,value);

// 读取
var score:string = egret.localStorage.getItem(key);

// 删除
egret.localStorage.removeItem(key);

// 清空所有数据
egret.localStorage.removeItem(key);


骨骼动画限制
帧动画前后必须有改动，否则会被当做无用帧清除，这样在程序中才能控制移动。
要用程序控制的动画必须要有改动，直接给ROOT添加一个动画就可以让所有动画可程序控制。

骨骼动画的offset坐标系中，使用的是左手坐标系。
而骨骼的global坐标系中，使用的是右手坐标系。

微信小游戏
有时候发布到微信小游戏的时候，皮肤会加载不出来。提示缺少xml解析库。
解决方法是：皮肤资源需要放在resource文件夹里面，放在这个文件夹下面，引擎会编译成类文件。
而放在其他其他文件夹下面，就不会自动编译导致微信中运行报缺少xml。

微信不支持show_all适配，要达到这种效果应该选用fixedNarrow适配。
然后在相关的视图上加上：

this.percentWidth = 100;
this.percentHeight = 100;

我这个是要在baseLayer和baseView里面加上。这个是居中操作，也可以使用其他居中操作。

有些时候编译后会有缓存，导致更改的代码不能生效，这时候应该清理项目或者使用浏览器来打开。

事件监听的添加应该方法资源加载之前，否则可能出现资源已经加载完成不触发监听事件的情况。
