对齐文本

对齐属性
+ text-align：指定文本块的对齐方式。
  其值：start、end、left、right、center。

代码：

  <style media="screen">
    [id|="t"]{
      display: inline-block;
      width: 100px;
      height: 100px;
      border: 2px solid black;
    }
    #t-start{ text-align: start; }
    #t-end{ text-align: end; }
    #t-left{ text-align: left; }
    #t-right{ text-align: right; }
    #t-center{ text-align: center; }
  </style>

  <div id="t-start">start</div>
  <div id="t-end">end</div>
  <div id="t-left">left</div>
  <div id="t-right">right</div>
  <div id="t-center">center</div>

处理空白

white-space属性
  其值：normal：默认值，空白符被压缩，文本行自动换行。
  　　　nowrap：空白符被压缩，本文行不换行。
    　　pre：空白符被保留，文本只在遇到换行符的时候换行，这跟pre元素的效果一样。
    　　pre-line：空白符被压缩，文本会在一行排满或遇到换行符时换行。
    　　pre-wrap：空白符被保留，文本会在一行排满或遇到换行符时换行。
