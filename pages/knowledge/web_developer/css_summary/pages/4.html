使用变换

+ transform                       指定应用于元素的变换。
+ transform-origin                指定元素变换的起点。

transform属性值
  + translate()                   在水平、垂直方向或两个方向上平移元素。
  + translateX()
  + translateY()
  + scale()                       在水平、垂直方向或两个方向上缩放元素。
  + scaleX()
  + scaleY()
  + rotate()                      旋转元素
  + skew()                        在水平、垂直方向或两个方向上倾斜元素。
  + skewX()
  + skewY()

transform-origin
  + %                       　     指定元素x轴或y轴的起点。
  + 长度值                  　     指定距离。
  + left、center、right            指定x轴的位置。
  + top、center、bottom            指定y轴上的位置。


使用过渡

过渡属性
+ transition                      指定CSS属性过渡效果的简写属性。
+ transition-delay                指定触发过渡的延迟时间。
+ transition-duration             指定过渡的持续时间。
+ transition-property             指定带有过渡效果的属性。
+ transition-timing-function      指定过渡期间计算中间属性值的函数。

transition简写属性的格式如下：
transition:transition-property transition-duration transition-timing-function transition-delay。

代码：

<style media="screen">
  #t0{
    background-color: red;
    font-size: 10px;
    color: white;
    transition-delay: 100ms;
    transition-duration: 1500ms;
    transition-property: background-color,color;
    transition-timing-function: linear;
  }
  #t0:hover{
    background-color: blue;
    font-size: 20px;
    color:black;
  }
</style>
<p id="t0">这是一段动画的测试文字。</p>

使用动画

+ @keyframes                      为动画指定一个以上的关键帧。
+ animation                       设置动画的简写属性。
+ animation-delay                 指定动画开始前的延迟时间。
+ animation-direction             指定动画重复播放时的播放方向。
  其值：normal、alternate。
+ animation-duration              指定动画持续时间。
+ animation-iteration-count       指定动画的循环次数。
  其值：infinite、具体数值。
+ animation-name                  指定用于动画的关键帧集合的名字。
+ animation-play-state            指定动画状态。
  其值：running、paused。
+ animation-timing-function       指定关键帧之间计算属性值的函数。

使用动画
动画停止播放后，他不会停止在当前的状态，而是初始化为最初的状态。
动画的使用分为两个部分。

第一部分：样式的声明。
第二部分：使用@key-frames创建具体规则

代码：

<style media="screen">
  #t1{
    width: 100px;
    height: 100px;
    animation-delay: 500ms;
    animation-direction: alternate;
    animation-duration: 2000ms;
    animation-timing-function: linear;
    animation-iteration-count: infinite;
    animation-name: t1,t1-1;
  }

  @keyframes t1 {
    /*指定初始值*/
    from{
      background-color: red;
    }
    /*指定某个阶段的值*/
    50%{
      background-color: blue;
    }
    /*指定最终值*/
    to{
      background-color: yellow;
    }
  }

  @keyframes t1-1 {
    50%{
      transform-origin: left top;
      transform: translateX(20px) translateY(10px) rotateX(90deg);
    }
    to{
      transform-origin: right bottom;
      transform: translateX(80px) translateY(-10px) rotateX(180deg);
    }
  }
</style>
