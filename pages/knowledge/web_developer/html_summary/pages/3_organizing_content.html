div元素


+ 局部属性：无
+ 示例：<div> ... </div>
+ 意义：无意义的块级元素


p元素


+ 局部属性：无
+ 示例：<p> ... </p>
+ 意义：段落


pre元素


+ 局部属性：无
+ 示例：<pre> ... </pre>
+ 意义：改变浏览器处理内容的方式，阻止合并空白字符，让源文档中的格式得以保留


blockquote元素


+ 局部属性：cite
+ 示例：<blockquote cite="http://"> ... </blockquote>
+ 意义：表示引自他处的一片内容。


hr元素


+ 局部属性：无
+ 示例：<hr>
+ 意义：表示段落级别的主题分隔


li元素


+ 局部属性：value（仅用于父元素为ol元素）
+ 示例：<li> ... </li>
+ 意义：表示列表中的项目

value属性可以用来指定某一项的编号。


ol元素


+ 局部属性：start、reversed、type
+ 意义：有序列表
+ 示例：

<ol>
     <li> ... </li>
</ol>


start属性设定的是列表首项的编号值，如果不用这个属性，那么首项的标号为1。
reversed属性使列表编号采用降序。
type属性用来设定显示在各列表项旁的标号的类型。

ol元素的type属性的值

+ 1: 十进制数（默认）
+ a：小写英文字母
+ A：大写英文字母
+ i：小写罗马数字
+ I：大写罗马数字


ul元素


+ 局部属性：无
+ 意义：无序列表
+ 示例：

<ul>
     <li> ... </li>
</ul>



dl、dt、dd元素


+ 局部属性：无
+ 意义：说明列表。
+ 示例：

<dl>
    <dt> ... </dt>
      <dd> ... <dd>
</dl>



figure元素


+ 局部属性：无
+ 意义：一个独立的内容单元，可带标题
+ 示例：

<figure>
    <figcaption> ... </figcaption>
    <p> ... </p>
</figure>
