使用

  2.运行文件
    进入cmd
    切换盘符    e:
    进入文件夹  cd node
    运行文件    node node.js

    代码：
    
      const http = require('http');

      var server = http.createServer(function(req,res)
      {
        //链接成功的回调函数
        console.log("链接");

        res.write("链接中");
        res.end();
      });

      //侦听
      server.listen(8080);
    


fs模块

  代码：
  
    const http = require("http");
    const fs = require("fs");

    var server = http.createServer(function(req,res)
    {
      var fileName = "/www" + req.url;

      //读文件
      fs.readFile("fileName",function(err,data)
      {
        if(err)
        {
          console.log(err);
        }
        else
        {
          res.write(data);
        }
        res.end();
      });

      //写文件
      //fs.writeFile("fileName","content",function(err){});
    });

    server.listen(8080);

        var str = "";

        {
          str += data;
        });

        //传送结束，一次

        {
          const POST = querystring.parse(str);

自定义模块

+ 自己的模块
  1.require     引入模块
  2.module      批量输出
  3.exports     导出数据

+ 引入模块
  如果自定义模块与自己定义的模块名称相同，系统模块优先。

  1.引入自己的模块的时候要加./
  2.将自定义的模块放入node_modules文件夹中，就不用加./。

+ .js是可选的
