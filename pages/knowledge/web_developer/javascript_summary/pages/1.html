+ 基本类型：string、number、boolean

  + 调用函数时提供的参数数目不必与函数定义中的参数数目相同。如果提供的参数值更少，那么所有未提供值的参数的
    值均为undefined。如果提供的参数值更多，那么多出的值会被忽略。

  + 要是定义了两个同名的函数，那么第二个定义将会取代第一个。

  + 对象的属性和方法可以用delete关键字删除。

  + 可以用in表达式判断对象是否具有某个属性。

  + ==运算符会尝试将操作数转换为同一类型以便判断是否相同。
    ===运算符会比较操作数的值和类型是否是都相同。

    基本类型比较的是值的比较，而引用类型是对地址的比较。

  + 数组内置方法：
    + pop()                    删除并返回最后一个元素
    + push(item)               将指定的元素添加到数目末尾
    + shift()                  删除并返回第一个元素
    + unshift(item)            将指定的元素添加到数目开头
    + concat(otherArray)       将使用的数组和指定的数组的内容合并成新的数组
    + join(separator)          将数组的内容以指定的分隔符转换为字符串
    + reverse()                反转数组
    + slice(start,end)         返回一个子串
    + sort()                   以小到大排序
