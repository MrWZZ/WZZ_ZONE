冒泡排序

   冒泡排序时可用的最慢的排序算法之一，但它是最容易理解和实现的一种排序算法。

   原理：
   1.选择第一个元素，令其与后面的元素进行比较，如果前面的值比后面的值大，则进行位置交换，选择位置前进一。
   2.依次比较到数组末尾。（这时，最大值的项已被放置到末尾，则下次需要比较的次数减一。）
   3.循环1、2步骤，直到最后只剩下一个元素。

   代码：
   
     var list = [2,4,1,3,7,5,0,8,9,6];
     function BubbleSort(sortList)
     {
       var temp;
       for(var i = sortList.length; i > 1; i--)
       {
         for(var j = 0; j < i - 1; j++)
         {
           if(sortList[j] > sortList[j+1])
           {
             temp = sortList[j];
             sortList[j] = sortList[j+1];
             sortList[j+1] = temp;
           }
         }
       }
     }
     BubbleSort(list);
