引入jar包

4+1：4个核心，一个依赖。
核心：bean、core、context、expression
依赖：commons-logging

配置文件
位置：任意，开发中在classpath(src)下。
名称：任意：开发中常用applicationCOntent
内容：约束、

  <?xml version="1.0" encoding="UTF-8"?>
  <beans xmlns="http://www.springframework.org/schema/beans"
      xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
      xsi:schemaLocation="http://www.springframework.org/schema/beans
          http://www.springframework.org/schema/beans/spring-beans.xsd">

      <bean id="..." class="...">
          <!-- collaborators and configuration for this bean go here -->
      </bean>

      <bean id="..." class="...">
          <!-- collaborators and configuration for this bean go here -->
      </bean>

      <!-- more bean definitions go here -->

  </beans>


使用静态工厂

<bean id="xxx" class="xxx" factory-method="静态工厂方法"></bean>


使用实例工厂

// 实例工厂bean
<bean id="beanF" class="xxx"></bean>

//通过实例工厂获取类
<bean id="xxx" factory-bean="beanF" factory-method="实例工厂中的方法"></bean>


生命周期：


+ B.before        [BeanPostProcessor]
- a.init()
+ B.after         [BeanPostProcessor]  //在这里生成代理

- a.addUser()

- a.destory()

初始化：init-method
销毁：destory-method。
  要求：必须close，单例。

BeanPostProcessor接口，接口中用两个方法，他将在初始化方法之前和之后调用特定方法。
