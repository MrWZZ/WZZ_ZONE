常用集合

List
list是一个有序的集合：每个元素都有位置。
Set
在set中，元素不会插到特定的位置，并且不允许出现重复的元素。
Queue
Queue会记住插入顺序，但是你只能在尾端插入元素，从头部删除他们。
Deque
双端队列有两个尾端，插入和删除都可以在两个尾部进行。
Map
类似C#的字典。


其他集合

Properties
用于存储配置文件，类似Unity的PlayerPref。

使用方法：

  //储存
  Properties settings = new Properties();
  settings.put("width","200");
  settings.put("title","HelloWorld!");
  try(OutputStream out = Files.newOutputStream(path))
  {
    settings.store(out,"Program Properties");
  }

  //读取
  try(InputStream in = Files.newInputStream(path))
  {
    settings.load(in);
  }
  String title = settings.getProperty("title","New Document");
