字符串比较
要检查两个字符串是否相等，使用equals方法。
不要使用==操作符比较字符串，在java虚拟机中，每一个字符串只有一个实例，因此"World"=="World"为真，但是，

  String location = greeting.substring(7,12);

  location == "World";    //为false



输入与输出

调用System.out.println时，输出发送到“标准输入流”，然后在窗口显示出来。
从“标准输入流”读取不是那样简单，因为相应的System.in对象只能读取单个字节的方法。
要读取字符串和数字，构造一个依附到System.in的Scanner。

  Scanner in = new Scanner(System.in);

  //nextLine方法从输入读取一行
  System.out.println("What is your name?");
  String name = in.nextLine();


格式化输出

  //表示输出的浮点数宽度为8，精度为小数点后两位，多余的以空格字符填充

数组列表
数组列表能自动扩充长度，不同于C#的是，Java没有索引器，不能使用[]直接读取数组列表中的某个值，需要使用其构造方法。

  //后面的<>中可以不填
  ArrayList<String> friends = new ArrayList<>();
  friends.add("Tom");
  String first = friends.get(0);
  firends.set(0,"Mary");


泛型类有个不幸的限制：不能使用基本类型作为类型参数。解决的方法是，使用包装类。
每一种类型都对应一种包装类：Integer、Byte、Short、Long、Character、Float、Double和Boolean。
而在这中方法中，自动装箱会自动完成。

数组列表的==比较的是引用，要比较值的话还是得使用equals()。


方法

静态方法：static

方法的命名开头是小写。

参数的传递都是值传递，没有ref、out。

可变参数

  //声明形式
  public static double average(double... values)
  {
    double sum = 0;
    for(double v : values) sum += v;
    return values.length == 0 ? 0 : sum/values.length;
  }
