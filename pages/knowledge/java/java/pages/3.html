接口

接口的关键字是interface。
接口的所有方法默认为共有方法。
接口的实现使用implements关键字。
实现多个接口时，用“，”分隔。

与c#不同的是，java的接口可以实现方法，拥有方法体。

静态方法


  public interface IntSequence
  {
    ...
    public static IntSequence digigtsOf(int n)
    {
      ...
    }
  }



默认方法

你可以给接口的任意方法提供默认实现，这需要default关键字。

  public interface IntSequence
  {
    default boolean hasNext()
    {
      ...
    }
  }


解决默认方法冲突
如果一个类实现了两个接口，且两个接口方法相同，其中一个或两个都拥有默认方法，此时，你必须解决冲突。

  public class Employee impements IPerson,IMan
  {
    public int getId()
    {
      return IMan.super.getId();
    }
  }


如果一个类继承一个父类，且实现了一个接口，如果有相同方法，接口的方法会被忽略。


Lambda表达式

与c#基本相似，不过他是用"->"来承接方法体。
